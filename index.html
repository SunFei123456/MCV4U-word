Function f(x) Derivative f'(x) Rule Name

Advanced Rules

Operation Formula Rule Name
f(x) · g(x) f'(x)g(x) + f(x)g'(x) Product Rule
f(x) / g(x) [f'(x)g(x) - f(x)g'(x)] / g²(x) Quotient Rule
f(g(x)) f'(g(x)) · g'(x) Chain Rule
aˣ aˣ · ln a General Exponential
logₐ x 1 / (x · ln a) General Logarithm

Limit Definitions

Concept Limit Expression Description
Derivative (Limit Definition) lim(h→0) [f(x+h) - f(x)] / h Standard Form
Derivative (Alternate) lim(x→a) [f(x) - f(a)] / (x - a) Point Form
Special Limit 1 lim(x→0) sin(x) / x = 1 Trigonometric
Special Limit 2 lim(x→0) (1 - cos x) / x = 0 Trigonometric
Euler's Limit lim(n→∞) (1 + 1/n)ⁿ = e Defines e

Real-World Applications

Field Formula Meaning
Kinematics v(t) = ds/dt Velocity is rate of change of position
Kinematics a(t) = dv/dt = d²s/dt² Acceleration is rate of change of velocity
Economics MC = dC/dx Marginal Cost
Economics MR = dR/dx Marginal Revenue
Optimization f'(c) = 0 Critical Point (find extrema)
